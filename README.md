# Another Authenticator
A two-factor authentication app built with Flutter.

## Development
`flutter run`

## Features
- Scan QR or manually input account details
- Supports both Material (Android) and Cupertino (iOS) themes

## Proposed
- View secrets
- Backup secrets to cloud storage
- UI tests
